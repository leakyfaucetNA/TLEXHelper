# Function


Read the Talent Loadout EX saved information to see which load out you are currently in.

When changing zone, if it is an instance it will check if you are in the correct spec.

TLEX shows a checkmark if your current talent loadout matches its loadout. 

The two intance options are Raid and Dungeon. 

The settings menu lets you define which specs are considered raid, and which are considered mythic plus. It does this with a dropdown with checkboxs for the loadouts.
